1) запустите Docker Desktop
2) запустить проект docker-compose up -d
3) зайти в браузер и ввести в адресную строку http://localhost/
4) если возникают какие-либо ошибки, еще раз ввести команду docker-compose up -d.
5) введите в форму имя, выберете категорию и нажмите отправить.
6) для того чтобы отследить успешное добавление в очередь, используйте консоль в браузере:
Google Chrome
Нажмите на клавиатуре F12 (или Fn + F12) или комбинацию Ctrl + Shift + I.
Яндекс
С помощью комбинации Ctrl + Shift + I.
Safari
Комбинация клавиш Command + Option + I.
7) буду рад обратной связи по замечаниям и предложениям по улучшению кода.
